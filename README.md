This is a **local-first** personal dashboard:
- Tasks (add/list/complete, priority + due date)
- Notes (quick capture + search)
- Habits (define habits + daily check-ins)
- Daily log (mood + energy + focus)
- Weekly report (auto summary)
- Export (Markdown report you can post/share)
- Local-first: **no server, no accounts**. Everything is stored in your browser (localStorage).

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
npm run preview
```

## Deploy (static host)
This is a Vite app. Build output is `dist/`.

- Build command: `npm run build`
- Output directory: `dist`
